## Getting started

With `node` > 16, install the deps with `npm i` and run it

```sh
# pass the filename of the badbits file you want to test with
node badbits.js badbits.3000.txt
```

If you want to get the latest, full, > 8MB denylist run

```
npm run denylist
```
